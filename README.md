# Large Folder Finder

Windows OS 上で大容量のフォルダーを高速に検索・特定するためのデスクトップアプリケーションです。

## 特徴

- **高速スキャン**: マルチスレッド処理により、ドライブ内の大容量フォルダーを迅速にリストアップします。
- **多言語対応**: 日本語、英語、ロシア語、イタリア語など、OSの言語設定に合わせて自動的に切り替わります。
- **表示カスタマイズ**: ファイルサイズの単位（GB, MBなど）や、出力形式（タブ区切り、スペース区切りなど）を詳細に設定可能です。
- **管理者権限対応**: アクセス権限のないシステムフォルダーなども、管理者権限で再起動することでスキャン可能になります。

## 使い方

1. `LargeFolderFinder.exe` を実行します。
2. スキャンしたいディレクトリを選択します。
3. 最小サイズ（例: 1GB以上）を設定し、「スキャン開始」をクリックします。
4. 結果が表示されます。右側の「コピー」ボタンで結果をクリップボードにコピーできます。

## システム要件

- Windows 10 / 11
- .NET 8.0 Runtime
