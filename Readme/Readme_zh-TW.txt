Large Folder Finder
====================
快速分析和列出資料夾層次結構的工具。
使用大小條件和篩選器(萬用字元、正規表示式)進行資料夾分析很有用。
有助於識別NAS等多人使用的大數據中的問題原因。


■ 使用方法
--------------------
  1. 選擇要調查的資料夾。
  2. 按"▶"(掃描)按鈕開始搜尋。
  3. 結果以類似Windows檔案總管的格式顯示。
  4. 指定顯示條件:要提取的最小大小、篩選器、排序、摺疊資料夾。
  5. 按右上角的複製按鈕將顯示結果複製到剪貼簿。
  6. 按索引標籤右側的"+"按鈕,在保留歷史記錄的同時開始新掃描。
    即使關閉應用程式,歷史記錄也會保留。
※ 以系統管理員權限執行應用程式可以分析C磁碟機內的系統管理員權限資料夾。
※ 可以從功能表/檢視切換語言或變更版面配置。
※ 可以從功能表/檢視/開啟進階設定(S)變更設定。詳細資訊如下所述。


■ 關於顯示功能
-------------------
1. 排序
  按一下每個標籤(名稱、大小、修改日期、類型)對顯示順序進行排序。
  再次按一下可在升序/降序之間切換。
2. 顯示檔案
  選取此項也會顯示檔案。
3. 最小大小
  指定要顯示的資料夾或檔案的最小大小。將顯示等於或大於設定值的項目。
  如果要顯示全部,請輸入0。
  單位可從Byte到TB選擇。
4. 篩選器
萬用字元:與Windows檔案總管相同的行為。
  * 允許符合任意字串。例) *.txt 所有名稱的txt檔案。例2) *資料* 名稱中包含"資料"的所有檔案。
  ? 允許符合任意單個字元。例) 202?年 → 2020年~2029年等。(也符合非數字)
  ~ 要搜尋(*或?)本身時,放在其前面。例) ~?.txt → 搜尋?.txt
正規表示式:進階篩選功能(工程師等使用)
  可以做萬用字元無法做到的事情。僅符合數字、小寫字母、大寫字母,僅提取不符合的項目等。
  由於複雜,請另外搜尋"如何使用正規表示式"。
  還有正規表示式檢查工具,可用於驗證搜尋是否正確運作。
5. 空格/定位字元
  按複製按鈕時,指定用空格還是定位字元填充名稱和大小之間的空間。


■ 無法正常運作時
------------------------
※ 可以從功能表/檢視/記錄檔檢查應用程式的行為。
※ 如果應用程式行為異常,刪除以下資料夾中的資料可能會重設快取並恢復功能。
    %LOCALAPPDATA%\Cat & Chocolate Laboratory\LargeFolderFinder


■ 關於進階設定 (Config.txt)
--------------------
透過編輯執行目錄中的"Config.txt",可以進行更詳細的行為設定。
按一下UI上的"⚙"按鈕可立即使用記事本等文字編輯器開啟它。
設定必須遵循YAML格式。如果要新增自己的註解,請在前面加上#。

    ▽ 可設定項目: (預設值)
    UseParallelScan: true
        類型: bool (true/false)
        說明: 啟用平行處理
        預期值 (true): 對NAS(網路儲存)等有效。本機SSD速度快,平行化開銷可能更大。

    SkipFolderCount: false
        類型: bool (true/false)
        說明: 是否跳過進度顯示的預先計數並立即開始掃描
        如果為true,則無法顯示進度百分比,因為總數未知。

    MaxDepthForCount: 3
        類型: int (自然數)
        說明: 為確定進度百分比而預先計數資料夾的最大層次深度
        指定的層次越大可能需要更多時間。但進度準確性會提高。
        預期值 (3): NAS: 3~6, PC內部: 7~

    UsePhysicalSize: true
        類型: bool (true/false)
        說明: 是否考慮叢集大小計算"磁碟上的配置大小"
        預期值 (true): 通常建議保持true。結果將更接近Windows內容顯示。如果為false,則按檔案大小計算。
        在調整此設定之前,建議以系統管理員身分執行。系統檔案也將包含在計算中以提高準確性。

    OldDataThresholdDays: 30
        類型：int (非負整數)
        說明：如果超過指定的天數，會將標籤頁高亮顯示為黃色，以表示掃描資料已過期。
        預期值：使用者偏好。

■ 如何新增語言檔案
--------------------
此工具支援多種語言,您可以新增新語言。
1. 開啟與應用程式可執行檔(.exe)相同層次結構中的"Languages"資料夾。
2. 複製現有的"en.yaml"等檔案,並將檔案名稱變更為要新增的語言的文化特性代碼(例如,法語為"fr.yaml")。
   * 有關文化特性代碼清單(例如: ja-JP / ja),請參閱以下內容:
   https://learn.microsoft.com/zh-tw/windows-hardware/manufacture/desktop/available-language-packs-for-windows?view=windows-11
3. 編輯YAML檔案中的文字(以UTF-8格式儲存)。
4. 重新啟動應用程式,新語言將顯示在"Language"功能表中。
※ 如有必要,請參考其他檔案建立並新增Readme_<language_code>.txt。


■ 完全解除安裝 (刪除設定和記錄檔)
--------------------
要完全刪除此工具的設定和執行記錄檔,請手動刪除以下資料夾:
%LOCALAPPDATA%\Cat & Chocolate Laboratory\LargeFolderFinder
(您可以透過將上述路徑貼上到檔案總管位址列中直接開啟它)
