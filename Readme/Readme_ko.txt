Large Folder Finder
====================
폴더 계층 구조를 빠르게 분석하고 나열하는 도구입니다.
크기 조건 및 필터(와일드카드, 정규식)를 사용하여 폴더 분석에 유용합니다.
NAS 등 여러 사용자가 사용하는 대용량 데이터에서 문제의 원인을 파악하는 데 도움이 됩니다.


■ 사용 방법
--------------------
  1. 조사하려는 폴더를 선택합니다.
  2. "▶" (스캔) 버튼을 눌러 검색을 시작합니다.
  3. 결과는 Windows 탐색기와 유사한 형식으로 표시됩니다.
  4. 표시 조건을 지정합니다: 추출할 최소 크기, 필터, 정렬, 폴더 접기.
  5. 오른쪽 상단의 복사 버튼을 눌러 표시 결과를 클립보드에 복사할 수 있습니다.
  6. 탭 오른쪽에 있는 "+" 버튼을 눌러 기록을 유지하면서 새 스캔을 시작할 수 있습니다.
    기록은 애플리케이션을 종료해도 유지됩니다.
※ 앱을 관리자 권한으로 실행하면 C 드라이브 내의 관리자 권한 폴더도 분석할 수 있습니다.
※ 메뉴/보기에서 언어를 전환하거나 레이아웃을 변경할 수 있습니다.
※ 메뉴/보기/고급 설정 열기(S)에서 설정을 변경할 수 있습니다. 자세한 내용은 아래를 참조하세요.


■ 표시 기능 정보
-------------------
1. 정렬
  각 레이블(이름, 크기, 수정 날짜, 유형)을 클릭하여 표시 순서를 정렬합니다.
  다시 클릭하면 오름차순/내림차순을 전환합니다.
2. 파일도 표시
  체크하면 파일도 표시됩니다.
3. 최소 크기
  표시할 폴더 또는 파일의 최소 크기를 지정합니다. 설정 값 이상의 항목이 표시됩니다.
  모두 표시하려면 0을 입력하세요.
  단위는 Byte에서 TB까지 선택할 수 있습니다.
4. 필터
와일드카드: Windows 탐색기와 동일한 동작입니다.
  * 임의의 문자열과 일치하도록 허용합니다. 예) *.txt 모든 이름의 txt 파일. 예2) *데이터* 이름에 "데이터"가 포함된 모든 파일.
  ? 임의의 한 문자와 일치하도록 허용합니다. 예) 202?년 → 2020년~2029년 등. (숫자 이외도 일치함)
  ~ (*나 ?) 자체를 검색하려면 앞에 배치합니다. 예) ~?.txt → ?.txt가 검색됨
정규식: 고급 필터 기능(엔지니어 등이 사용)
  와일드카드로 할 수 없는 것도 할 수 있습니다. 숫자, 소문자, 대문자만 일치, 일치하지 않는 것만 추출 등.
  복잡하므로 별도로 "정규식 사용법"을 검색해 보세요.
  정규식 검사 도구도 있으니 올바르게 검색되는지 활용해 보세요.
5. 공백/탭
  복사 버튼을 눌렀을 때 이름과 크기 사이를 채우는 문자열을 공백 또는 탭으로 지정합니다.


■ 제대로 작동하지 않을 때
------------------------
※ 메뉴/보기/로그에서 앱의 동작을 확인할 수 있습니다.
※ 앱이 이상하게 작동하면 다음 폴더의 데이터를 삭제하면 캐시가 재설정되어 작동할 수 있습니다.
    %LOCALAPPDATA%\Cat & Chocolate Laboratory\LargeFolderFinder


■ 고급 설정 정보 (Config.txt)
--------------------
실행 디렉터리에 있는 "Config.txt"를 편집하여 더 자세한 동작 설정이 가능합니다.
UI의 "⚙" 버튼을 클릭하면 메모장 등의 텍스트 편집기로 즉시 열 수 있습니다.
설정은 YAML 형식을 따라야 합니다. 자체 주석을 추가하려면 #을 앞에 붙이세요.

    ▽ 설정 가능한 항목: (기본값)
    UseParallelScan: true
        유형: bool (true/false)
        설명: 병렬 처리 활성화
        예상 값 (true): NAS(네트워크 스토리지)에서 효과적입니다. 로컬 SSD는 빠르므로 병렬화 오버헤드가 더 클 수 있습니다.

    SkipFolderCount: false
        유형: bool (true/false)
        설명: 진행률 표시를 위한 사전 카운트를 건너뛰고 즉시 스캔을 시작할지 여부
        true인 경우 총 수를 알 수 없으므로 진행률이 표시되지 않습니다.

    MaxDepthForCount: 3
        유형: int (자연수)
        설명: 진행률을 파악하기 위해 폴더 총 수를 사전 카운트하는 최대 계층
        지정한 계층이 크면 시간이 걸릴 수 있습니다. 대신 진행률 정확도가 향상됩니다.
        예상 값 (3): NAS: 3~6, PC 내부: 7~

    UsePhysicalSize: true
        유형: bool (true/false)
        설명: 클러스터 크기를 고려하여 "디스크의 할당된 크기"를 계산할지 여부
        예상 값 (true): 일반적으로 true 권장. Windows 속성 표시에 가까운 값이 됩니다. false이면 파일 크기로 계산됩니다.
        이 설정을 조정하기 전에 관리자 권한으로 실행하는 것이 좋습니다. 시스템 파일도 포함하여 계산되어 정확해집니다.

    OldDataThresholdDays: 30
        유형: int (음수가 아닌 정수)
        설명: 지정된 일수가 지나면 스캔 데이터가 오래되었음을 나타내기 위해 탭을 노란색으로 강조 표시합니다.
        예상 값: 사용자 선호.

■ 언어 파일 추가 방법
--------------------
이 도구는 다국어를 지원하며 새 언어를 추가할 수 있습니다.
1. 앱 실행 파일(.exe)과 같은 계층에 있는 "Languages" 폴더를 엽니다.
2. 기존 "en.yaml" 등을 복사하여 추가하려는 언어의 문화 코드(예: 프랑스어의 경우 "fr.yaml")로 파일 이름을 변경합니다.
   * 문화 코드(예: ja-JP / ja) 목록은 다음을 참조하세요:
   https://learn.microsoft.com/ko-kr/windows-hardware/manufacture/desktop/available-language-packs-for-windows?view=windows-11
3. YAML 파일 내의 텍스트를 편집합니다(UTF-8 형식으로 저장하세요).
4. 앱을 다시 시작하면 "Language" 메뉴에 새 언어가 표시됩니다.
※ 필요에 따라 다른 파일을 참조하여 Readme_<language_code>.txt도 작성하여 추가하세요.


■ 완전 제거 (설정 및 로그 삭제)
--------------------
이 도구의 설정 및 실행 로그를 완전히 제거하려면 다음 폴더를 수동으로 삭제하세요:
%LOCALAPPDATA%\Cat & Chocolate Laboratory\LargeFolderFinder
(탐색기 주소 표시줄에 위 경로를 붙여넣으면 직접 열 수 있습니다)
