Large Folder Finder
====================
지정한 크기 이상의 폴더를 신속하게 추출하여 목록으로 표시하는 도구입니다.


■ 사용 방법
--------------------
1. 조사하고자 하는 폴더를 선택합니다.
2. 추출하고자 하는 최소 크기를 지정합니다.
3. "Scan" 버튼을 누르면 탐색이 시작됩니다.
4. 결과는 텍스트 형식으로 표시됩니다.
5. 오른쪽 상단의 복사 버튼(📄 아이콘)을 누르면 결과를 클립보드에 복사할 수 있습니다.


■ 고급 설정에 대하여 (Config.txt)
--------------------
실행 디렉터리에 있는 "Config.txt"를 편집하여 더욱 상세한 동작 설정을 할 수 있습니다.
UI 상의 "⚙" 버튼을 클릭하면 메모장 등의 텍스트 에디터로 즉시 열 수 있습니다.
설정 방식은 Yaml 형식을 따라야 합니다. 직접 코멘트를 작성하고 싶은 경우, 행 앞에 #을 붙여주세요.

    ▽ 설정 가능한 항목: (기본값)
    UseParallelScan: true
        형식: bool (true/false)
        설명: 병렬 스캔 활성화 여부
        상세 (true): NAS(네트워크 스토리지) 등에서 효과를 발휘합니다. 로컬 SSD는 고속이기 때문에 병렬화의 오버헤드가 더 클 수 있습니다.

    SkipFolderCount: false
        형식: bool (true/false)
        설명: 진행률 표시용 사전 카운트를 건너뛰고 즉시 스캔을 시작할지 여부
        true로 설정하면 폴더 총수를 알 수 없으므로 진행률(%)이 표시되지 않습니다.

    MaxDepthForCount: 3
        형식: int (자연수)
        설명: 진행률을 파악하기 위해 사전에 폴더 총수 카운트를 수행할 최대 계층 깊이
        값이 클수록 시간이 더 걸릴 수 있지만 진행률의 정확도는 올라갑니다.
        예시 (3): NAS: 3~6, 사내 PC: 7~

    UsePhysicalSize: true
        형식: bool (true/false)
        설명: 클러스터 크기를 고려하여 '디스크 할당 크기'를 계산할지 여부
        예시 (true): 보통 true를 권장합니다. Windows의 속성 표시값에 가까운 결과가 나옵니다. false일 경우 실제 파일 크기로 계산됩니다.
        이 설정을 조정하기 전에 앱을 관리자 권한으로 실행하는 것을 권장합니다. 시스템 파일까지 정확하게 계산에 포함됩니다.


■ 언어 파일 추가 방법
--------------------
본 도구는 다국어를 지원하며, 새로운 언어를 추가할 수 있습니다.
1. 앱 실행 파일(.exe)과 같은 계층에 있는 "Languages" 폴더를 엽니다.
2. 기존의 "en.yaml" 등을 복사하여, 파일명을 추가하고자 하는 언어의 문화권 코드(예: 프랑스어라면 "fr.yaml")로 변경합니다.
   * 문화권 코드 목록은 다음 Microsoft 문서를 참조하십시오:
   https://learn.microsoft.com/ko-kr/windows-hardware/manufacture/desktop/available-language-packs-for-windows?view=windows-11
3. YAML 파일 내의 텍스트를 편집합니다(UTF-8 형식으로 저장해 주십시오).
4. 앱을 재기동하면 메뉴의 "Language"에 새로운 언어가 표시됩니다.
* 필요에 따라 다른 파일을 참고하여 Readme_<code>.txt를 작성하고 추가하십시오.


■ 완전 설치 제거 (설정·로그 삭제)
--------------------
본 도구의 설정이나 실행 로그를 완전히 삭제하려면 다음 폴더를 수동으로 삭제해 주십시오.
%LOCALAPPDATA%\Cat & Chocolate Laboratory\LargeFolderFinder
(파일 탐색기의 주소창에 위 경로를 붙여넣으면 직접 열 수 있습니다)
