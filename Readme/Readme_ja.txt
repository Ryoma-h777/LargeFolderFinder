Large Folder Finder
====================
指定したサイズ以上のフォルダを高速に抽出して一覧表示するツールです。


■ 使い方
--------------------
1. 調査したいフォルダを選択します。
2. 抽出したい最小サイズを指定します。
3. 「Scan」ボタンを押すと探索が始まります。
4. 結果はテキスト形式で表示されます。
5. 右上のコピーボタン（📄アイコン）を押すと、結果をクリップボードにコピーできます。


■ 高度な設定について (Config.txt)
--------------------
実行ディレクトリにある「Config.txt」を編集することで、より詳細な動作設定が可能です。
UI上の「⚙」ボタンをクリックすることで、メモ帳などのテキストエディタで即座に開くことができます。
設定方法はYaml形式に則り行ってください。独自でコメントを記載したい場合は、#を先頭につけてください。

    ▽設定可能な項目：(既定値)
    UseParallelScan: false
        型：bool (true/false)
        説明：並列処理の有効化
        想定値 (false): HDD(NAS)は回転する都合上並列に弱いため false に設定。SSDのみ true 推奨。

    SkipFolderCount: false
        型：bool (true/false)
        説明：進捗率表示用の事前カウントをスキップして即座にスキャンを開始するか
        true の場合は総数を把握できないため、進捗率は表示されません。

    MaxDepthForCount: 3
        型：int (自然数)
        説明：進捗率を把握するために、事前にフォルダ総数カウントを行う最大階層
        指定した階層が大きいと時間がかかる可能性があります。代わりに進捗率の精度が上がります。
        想定値 (3): NAS: 3~6, PC内 : 7~

    UsePhysicalSize: true
        型：bool (true/false)
        説明：クラスタサイズを考慮して「ディスク上の占有サイズ」を計算するかどうか
        想定値 (true): 通常は true 推奨。Windowsのプロパティ表示に近い値になります。falseだとファイルのサイズで計算されます。
        この設定を見直す前に、管理者権限で実行することをお勧めします。システムファイルも含めて計算され正確になります。


■ 言語ファイルの追加方法
--------------------
本ツールは多言語に対応しており、新しい言語を追加することが可能です。
1. アプリ実行ファイル（.exe）と同じ階層にある「Languages」フォルダを開きます。
2. 既存の「en.yaml」などをコピーし、ファイル名を追加したい言語のカルチャーコード（例：フランス語なら「fr.yaml」）に変更します。
   ※カルチャーコード（例: ja-JP / ja）の一覧は以下を参照してください。
   https://learn.microsoft.com/ja-jp/windows-hardware/manufacture/desktop/available-language-packs-for-windows?view=windows-11
3. YAMLファイル内のテキストを編集します（UTF-8形式で保存してください）。
4. アプリを再起動すると、メニューの「Language」に新しい言語が表示されます。
※必要に応じて、Readme_<code>.txtも同様に他のファイルを参考に作成して追加してください。


■ 完全アンインストール（設定・ログの削除）
--------------------
本ツールの設定や実行ログを完全に削除する場合は、以下のフォルダを手動で削除してください。
%LOCALAPPDATA%\Cat & Chocolate Laboratory\LargeFolderFinder
（エクスプローラーのアドレスバーに上記パスを貼り付けると直接開けます）
